Mit Mausklick auf den entsprechenden Admin-Tab den Arbeitsbereich {PLUGINS_BUTTON} aufrufen.

{MODUL_mozilo_tabs_plugins}

In diesem Bereich werden Plugins hinzugefügt oder entfernt, sowie sämtliche Einstellungen innerhalb der Plugins vorgenommen.

Plugins verwalten

1Durch Klick auf das Werkzeug-Symbol {MO_ICON_edit} wird die Plugin-Verwaltung ausgeklappt. In diesem Bereich werden Plugins installiert oder gelöscht.

{MODUL_plugins_00}
1

Plugins installieren:

1Das zu installierende Plugin auf dem Datenträger suchen und auswählen.

2Durch Klick auf "Install" wird das Plugin hochgeladen und installiert.

Plugins müssen als gepacktes .zip Archiv vorliegen um erkannt und installiert zu werden.
Es kann auch eine ZIP Datei mit mehren Plugins sein z.B. aus einer Backup Datei nur mit Plugins.

Optional kann die ZIP Datei auch in den "plugins" Ordner gelegt werden.
Dann erscheint so eine Auswahl {MODUL_plugins_01a}

Falls nach der Installation nur noch eine weiße Seite erscheint muss das Plugin per FTP entfernt werden.
Es sollte dann der Plugin Autor benachrichtigt werden, dass sein Plugin fehlerhaft ist.

{MODUL_plugins_01}
1
2

Plugins löschen:

1Durch Klick auf den Button "Ausgewählte Plugins löschen" werden alle Plugins gelöscht bei denen die rechte Checkbox aktiviert ist.

2Checkbox aktivieren um Plugin zum Löschen freizugeben.

{MODUL_plugins_02}
1
2

Plugin-Liste

In der Grundinstallation von moziloCMS sind bereits einige Plugins enthalten. Jedes installierte Plugin erscheint in dieser Liste als separater Balken.

1Die installierten Plugins können durch setzen des Häkchens in der Checkbox aktiviert oder deaktiviert werden.

{MODUL_plugins_03}
1

Plugins Filter:

1Bei Eingabe eines Buchstaben oder Namensteils bleiben in der Liste nur diejenigen Plugins sichtbar, in deren Namen dieser Buchstabe oder Namensteil enthalten ist. Alle anderen werden ausgeblendet.

{MODUL_plugins_04}
1

Plugin Einstellungen:

1Durch Klick auf das Werkzeug-Symbol {MO_ICON_edit} im Plugin-Balken öffnet sich der Konfigurationsbereich des jeweiligen Plugins.

2Durch Klick auf das Info-Symbol {MO_ICON_info} öffnet sich eine Beschreibung des Plugins und seiner Konfiguration

3Einstellungen müssen durch Klick auf das Speichern-Symbol {MO_ICON_save} gespeichert werden
